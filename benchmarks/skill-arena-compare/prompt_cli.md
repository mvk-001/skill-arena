Trabaja en un loop autónomo para mejorar el cli.

Reglas:
- En cada iteración, prueba solo una hipótesis.
- En cada iteracion solo usa una skill para evaluar
- No repitas skills

Loop:
1. Selecciona una skill desde http://github.com/obra/superpowers/tree/main/skills
2. Intenta crear el archivo de comparación solo con el cli
3. Si no puedes busca como mejorarlo para poder realizarlo.
4. Si puedes, ejecuta la comparacion con 1 solo request
5. Si no funciona mejora el cli en lo que necesite
6. Solo usa una skill a la vez para evolucionar el cli
8. Append a cli_evolve.log
9. Repite
